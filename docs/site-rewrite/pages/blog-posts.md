# Blog Posts

Статус: draft
Маршрут: `/blog/:slug`

## Цель документа

Этот файл нужен для правил переписывания отдельных статей блога.

## Общие правила для статей

Каждая статья должна содержать:
- тему;
- целевую аудиторию;
- цель статьи;
- ключевой поисковый запрос;
- H1;
- description;
- структуру по блокам;
- CTA в конце статьи.

## Шаблон карточки статьи

